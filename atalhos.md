

# Kitty

- Split window: `Ctrl + Shift + Enter`
- Próxima janela: `Ctrl + Shift + ]`
- Janela anterior: `Ctrl + Shift + [`


# VSCode

- Ir para o Explorer: `Ctrl + Shift + e`

# Navegador

- Voltar no hitórico: `H`
- Avançar no histórico: `L`
- Editar endereço: `ge`
